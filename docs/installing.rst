Installing Pyrox
================

Debian Compatible Systems
-------------------------

Download the correct .deb file for your architecture.

* Debian Wheezy
* Ubuntu 12.04 LTS

::

    curl --remote-name http://pyrox-http.org/deb/<distro>/pyrox_<version>_<arch>.deb
    dpkg -i pyrox_<version>_<arch>.deb
    service pyrox start
